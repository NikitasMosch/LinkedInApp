Welcome

Through the management page, the admin can navigate to the list of users. From the user list, the admin can navigate to each user's detail page, where they can examine the user's information. Additionally, the admin has the ability to export user data in XML and JSON formats by selecting one or more users. This includes their resume, article and job postings, professional experience, notes of interest, comments on other professionals' articles, and their network of connected professionals.
